# Український Освітній Хаб
### Пропозиція з розробки веб-сайту

---

## Що буде створено

Сучасний, адаптивний веб-сайт для презентації мережі Українських Освітніх Хабів у Великій Британії.

### Основний функціонал:

| Елемент | Опис |
|---------|------|
| Головна сторінка | Місія, напрямки, новини |
| Інтерактивна карта UK | Клік по місту → сторінка хабу |
| 7 сторінок хабів | Лондон, Оксфорд, Чічестер, Манчестер, Бат, Лідс, Челтнем |
| Фото-галереї | Слайд-шоу, повноекранний режим, до 100 фото на хаб |
| Адмін-панель (CMS) | Редагування текстів, фото, додавання нових хабів |
| Footer з партнерами | Логотипи, соцмережі, контакти |

### Технічні характеристики:
- Адаптивний дизайн (телефон, планшет, комп'ютер)
- Швидке завантаження (до 2-3 секунд)
- SEO-оптимізація
- SSL-сертифікат
- Масштабування до 20+ хабів

## Орієнтовні терміни

| Етап | Термін |
|------|--------|
| Прототип (структура + карта) | 2-3 дні |
| Дизайн + верстка всіх сторінок | 3-4 дні |
| Галереї + CMS + адмінка | 2-3 дні |
| Наповнення контентом | 2-3 дні* |
| Тестування + правки | 1-2 дні |
| **Загалом** | **~2 тижні** |

*\* Залежить від швидкості надання матеріалів*

## Що потрібно від кожної сторони

### Від організації:

- [ ] Тексти опису для кожного хабу (7 міст)
- [ ] Фотографії (50-100 на хаб, бажано в хорошій якості)
- [ ] Логотипи партнерів
- [ ] Контактна інформація по кожному місту
- [ ] Посилання на соцмережі
- [ ] Оплата домену (~£10-15/рік)
- [ ] Бажане доменне ім'я (наприклад: ukrainianeduhub.co.uk)

### Від розробника:

- [ ] Повна розробка сайту
- [ ] Дизайн
- [ ] Адмін-панель
- [ ] Оптимізація та тестування
- [ ] Навчання як користуватись CMS
- [ ] Дрібні правки після запуску

## Умови співпраці

Розробка виконується **безкоштовно** в обмін на:

| Що | Коли |
